Datasets used in this project

Greenhouse Gas Emissions

Source: UNFCCC (United Nations Framework Convention on Climate Change)

This dataset contains country-level greenhouse gas emissions data collected by the UNFCCC. For this projects, certain years were used to match other datasets. Countries that have missing values were removed to ensure the consistency across datasets.

Link: GHG data from UNFCCC

Gross Domestic Product (GDP)

Source: World Bank - World Development Indicators (WDI)

This dataset provides GDP data for countries worldwide. The data was filtered to include only the years and countries that overlap with the Greenhouse Gas Emissions dataset. Country name was standardized to ensure accurate merging with other datasets.

Link: GDP data from WorldBank

Population

Source: UN Data / World Bank WDI

This dataset contains population data for each country. It was used to calculate GDP per capita and greenhouse gas per capita. The dataset was cleaned and aligned with GDP and Greenhouse Gas Emissions datasets to ensure consistency.

Link: Population data from WorldBank
